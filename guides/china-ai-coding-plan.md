# 国产 AI Coding Plan 目录：Kimi、MiniMax、GLM、Trae 综合对比

> 可视化页面：[https://codingplan.pw/china-ai-coding-plan/](https://codingplan.pw/china-ai-coding-plan/)

这个专题整理国内用户更容易注册、支付和访问的 AI Coding Plan。重点比较价格、额度、额度重置、优惠、官网入口、Claude Code 平替能力、Cursor 接入和团队落地成本。

## 快速对比

| 产品 | 类型 | 入门价格 | 主要能力 | 重点关注 |
|---|---|---:|---|---|
| Kimi Coding Plan | 国产 Coding Plan | ¥49/月 | Kimi Code、长上下文、Claude Code 接入 | 中文体验、代码生成、Agent |
| MiniMax Coding Plan | 国产模型 / Agent | ¥49/月 | MiniMax-M3 / M2.7、多模态、工具调用 | Agent、多模态、额度池 |
| Zhipu GLM Coding Plan | 国产 Coding Plan | ¥49/月 | GLM 系列、代码、多模态 | 企业生态、国产模型接入 |
| Trae Coding Plan | AI IDE | $10/月 | Agent、补全、多模型 | 低门槛 AI IDE、国内用户 |
| Qwen / Doubao API | 国产 API | 按量计费 | API 接入、企业应用 | 自建 Coding Agent、工具链接入 |

## 选择维度

| 维度 | 说明 |
|---|---|
| 价格 | 月费、人民币支付、API 按量、团队套餐 |
| 额度 | 高级模型次数、总额度、额度重置、不同模型消耗倍率 |
| 接入 | Claude Code、Cursor、自定义 API、OpenAI 兼容程度 |
| 代码能力 | 多文件修改、中文项目理解、测试修复、框架熟悉度 |
| 国内体验 | 注册、支付、访问、控制台、发票和企业认证 |
| 合规 | 数据策略、团队权限、是否适合公司代码库 |

## 推荐场景

### 个人开发者

优先看 Kimi、Trae、MiniMax。重点看月费、可用额度、中文体验和是否能接入现有编辑器。

### 公司内部工具

优先看 GLM、Qwen、Doubao、MiniMax。重点看企业认证、API 稳定性、发票、权限和私有代码策略。

### Claude Code 平替

优先看 Kimi、MiniMax、GLM。重点验证 base URL、模型名、工具调用、上下文长度和额度消耗。
