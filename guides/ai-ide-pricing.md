# AI IDE 套餐价格目录：Cursor、Trae、Windsurf、Copilot 综合对比

> 可视化页面：[https://codingplan.pw/ai-ide-pricing/](https://codingplan.pw/ai-ide-pricing/)

这个专题不是单纯列价格，而是把 AI IDE 的价格、额度、Agent 能力、自动补全、模型选择、优惠、官网入口和团队协作放在一起对比。

数据来源：[TokenPlan 详情：https://tokenplan.vip/](https://tokenplan.vip/)

## 快速对比

| 产品 | 入门价格 | 类型 | 主要能力 | 重点关注 | 适合人群 |
|---|---:|---|---|---|---|
| Cursor Pro | $20/月 | AI IDE | Composer / Agent / 补全 | IDE 内深度集成、多模型 | 日常开发、重度 AI 编程 |
| Trae Pro | $10/月 | AI IDE | Agent / 补全 / 多模型 | 低门槛、价格友好 | 个人开发者、国内用户 |
| Windsurf Pro | $20/月 | AI IDE | Cascade / Agent | 多步骤任务、编辑器工作流 | 项目级重构、Agent 开发 |
| GitHub Copilot Pro | $10/月 | 编程助手 | 补全 / Chat / Agent Mode | GitHub 生态、VS Code 集成 | GitHub 用户、团队协作 |

## 对比维度

| 维度 | 说明 |
|---|---|
| 价格 | 月费、年付、团队版、学生优惠和地区支付 |
| 额度 | 高级模型次数、补全额度、Agent 调用限制、是否重置 |
| 模型 | 是否支持 Claude、GPT、Gemini、国产模型或自定义 API |
| Agent | 是否能读写多文件、执行命令、运行测试、处理大任务 |
| 生态 | VS Code 兼容、GitHub 集成、插件、远程开发支持 |
| 国内体验 | 访问速度、支付方式、账号注册和网络稳定性 |

## 推荐场景

### 想要最完整 AI IDE

优先比较 Cursor 和 Windsurf。重点看 Agent 是否稳定、上下文是否够用、项目级修改体验如何。

### 想低成本体验 AI IDE

优先看 Trae 和 GitHub Copilot。重点看月费、补全体验、模型额度和国内访问。

### 团队协作

重点比较 GitHub Copilot、Cursor Teams、Windsurf Teams。看权限、成员管理、审计、企业数据策略。
